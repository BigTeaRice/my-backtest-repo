# 📊 多策略回测系统

一个完整的自动化多策略回测系统，支持五种技术指标策略，每天自动运行回测并生成报告。

## ✨ 功能特性

- **五种策略支持**：SMA双均线、RSI超买超卖、MACD交叉、布林带、KDI随机指标
- **自动化回测**：通过GitHub Actions每天自动运行回测
- **完整报告**：生成HTML、JSON格式的详细回测报告
- **策略对比**：自动比较不同策略在不同标的上的表现
- **可视化界面**：友好的Web界面查看结果
- **数据缓存**：本地缓存机制加速回测过程
