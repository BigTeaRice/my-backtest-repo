# 📊 多策略回测系统

一个基于 GitHub Actions 自动运行的多策略回测系统，支持 SMA、RSI、MACD、布林带、KDJ 五种技术指标策略。

## 🚀 功能特点

- **5大技术指标策略**：
  - SMA双均线策略
  - RSI超买超卖策略  
  - MACD交叉策略
  - 布林带策略
  - KDJ随机指标策略

- **多市场支持**：
  - 港股：恒生指数、腾讯、阿里等
  - 美股：苹果、微软、谷歌、特斯拉、英伟达等
  - ETF：标普500 ETF、纳指100 ETF

- **自动运行**：
  - 每天自动运行回测
  - 自动生成图表和报告
  - 自动部署到 GitHub Pages
