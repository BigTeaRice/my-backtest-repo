# 📊 多策略回测系统

一个基于 GitHub Actions 自动运行的多策略回测系统，支持 SMA、RSI、MACD、布林带、KDJ 五种技术指标策略。

## 🌐 在线演示

访问地址: https://[你的用户名].github.io/[仓库名称]/

## 🚀 功能特点

### 📈 5大技术指标策略
- **SMA策略**: 双均线交叉策略
- **RSI策略**: RSI超买超卖策略  
- **MACD策略**: MACD金叉死叉策略
- **布林带策略**: 布林带上下轨策略
- **KDJ策略**: KDJ随机指标策略

### 🌍 多市场支持
- **美股**: 苹果、微软、谷歌、特斯拉、英伟达等
- **港股**: 腾讯、阿里、美团、小米等
- **指数**: 标普500、纳斯达克、道琼斯、恒生指数

### 📊 完整回测报告
- 交互式图表 (Plotly.js)
- 详细性能指标
- 交易统计数据
- 风险分析指标

### ⚙️ 自动运行
- 每天 UTC 22:00 自动更新
- GitHub Pages 自动部署
- 手动触发支持
